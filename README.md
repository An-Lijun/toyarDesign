## 安装

```
    npm i toyar-design
```

## 使用

```JavaScript

//main.js

import toyar from 'toyar-design/dist'
import 'toyar-design/dist/style.css'

createApp(App)
    .use(store)
    .use(toyar)
    .use(router)
    .mount('#app')

```
