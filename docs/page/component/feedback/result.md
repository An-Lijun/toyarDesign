## 属性(Attributes)


| 属性       | 描述                           | 类型     | 值                        | 默认          | 
| ---------- | ------------------------------ | -------- | ------------------------- | ------------- |
| size       | 组件图标大小                       | string   | -- | 100 |
| type       | 结果组件的类型                       | string   | -- |  info/success/error/warning  |
